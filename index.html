HTML TAGS

Importance is due to semantic tags help with SEO and screen readers so accessiblity as well

Header

Header tag signifies the 'header' of your page - typically when you want to put an importance such as a headline.

Main

Main tag sepcifies content inside the

element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.

Footer

The Footer tag defines a footer for a document or section. A Footer element typically contains:
  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents

You can have several

elements in one document.

Article

The Article element specifies independent, self-contained content.

An Article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.

Nav

Nav element defines a set of navigation links.

Sidebar

Sidebar element defines some content aside from the content it is placed in (like a sidebar)
